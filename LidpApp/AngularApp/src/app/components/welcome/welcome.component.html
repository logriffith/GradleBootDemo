LIDP UI Challenge

This application consists of coding challenges. These challenges can be summarized below. Feel free to interact with each challenge via the navigation bar.

Challenges:

  • Hello World: a user can type their name and see it displayed on the page with a greeting as they type. String interpolation and two-way binding were used for this challenge.
  • GET: a GET request is sent when the page loads to retrieve today's date and name of the application from a web service. This information is displayed at the top of the page.
  • POST: a user can calculate the circumference of a circle by sending a radius via a POST request.
  • Material:
  • System Stats
  • Database: store a user's favorite colors in a PostreSQL database. A user can create, retrieve, update, and delete their info.
